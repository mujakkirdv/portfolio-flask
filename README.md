# Mujakkir Ahmad - Professional Data Analyst Portfolio

A comprehensive, interactive portfolio website showcasing data analytics expertise with advanced 3D visualizations, business intelligence dashboards, and professional experience.

## 🚀 Live Demo

[View Live Portfolio](#) - *Coming soon*

## 📊 Features

### 🎨 Advanced 3D Visualizations
- **3D Surface Plots**: Marketing spend vs customer engagement analysis
- **3D Scatter Plots**: Customer segmentation and behavior patterns
- **3D Mesh Networks**: Product performance matrices
- **3D Skill Proficiency**: Interactive skill assessment visualizations
- **3D Network Graphs**: Professional connection mapping
- **3D Geographic Mapping**: Regional engagement analysis

### 📈 Business Intelligence Dashboards
- **Sales Performance Tracking**: Real-time sales analytics
- **Customer Segmentation**: Age, income, and spending pattern analysis
- **Product Performance**: Profit margins and satisfaction metrics
- **ROI Analysis**: Project impact and investment returns
- **Correlation Analysis**: Business metric relationships

### 💼 Professional Portfolio
- **Work Experience**: Detailed career history with achievements
- **Project Gallery**: Showcase of data analytics projects
- **Technical Skills**: Interactive skill proficiency visualization
- **Contact Form**: Direct communication channel
- **Professional References**: Verified professional contacts

## 🛠️ Technology Stack

### Backend
- **Python 3.12.0**: Core programming language
- **Flask 3.0.0**: Web framework
- **Plotly 5.18.0**: Interactive 3D visualizations
- **Pandas 2.1.3**: Data manipulation
- **NumPy 1.26.2**: Numerical computing
- **SciPy 1.11.4**: Scientific computing

### Frontend
- **HTML5/CSS3**: Structure and styling
- **Bootstrap 5**: Responsive design
- **JavaScript**: Interactive elements
- **Font Awesome 6**: Icons and graphics

### Development Tools
- **VS Code**: IDE
- **Git**: Version control
- **GitHub**: Repository hosting

## 🚀 Installation Guide

### Prerequisites
- Python 3.12.0 or higher
- pip (Python package manager)
- Git (optional)

### Step-by-Step Installation

1. **Clone the repository**
```bash
git clone https://github.com/mujakkirdv/portfolio-flask.git
cd portfolio-flask


2. Create virtual environment

# Windows
python -m venv venv
venv\Scripts\activate

# macOS/Linux
python3 -m venv venv
source venv/bin/activate

Install dependencies
pip install -r requirements.txt

Add images
Place your images in the static/images/ directory:

profile.jpg - Professional photo (600x600)

logo.png - Site logo (512x512)

project1.png, project2.png, ai-dashboard.png - Project screenshots (800x600)
